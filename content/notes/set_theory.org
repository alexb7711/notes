* Set Theory
:PROPERTIES:
:CUSTOM_ID: set-theory
:END:
A set is an unordered collection of different elements. For example

\[
A = \{1,2,3,4,5\}
\]

is a set #1.

** Representation of Sets
:PROPERTIES:
:CUSTOM_ID: representation-of-sets
:END:
Sets are typically represented in two ways

- Roster or Tabular Form

\[
A = \{1,2,3,4\}
\]

- Set Builder Notation

\[
A = \{x:p(x)\}
\]

where \(p(x)\) is some function that describes the set
#1.

** Types of Sets
:PROPERTIES:
:CUSTOM_ID: types-of-sets
:END:
The following are definitions of some basic sets
#1:

- Finite Set: \(S = \{x | x \in N and 70 > x > 50\}\)
- Infinite Set: $ S = {x | x \in N and x > 50}$
- Subset: \(S = X \subseteq Y\)
- Proper Subset: \(S = X \subset Y\)
- Universal Set: The collection of all elements in a particular context
  or application
- Empty (Null) Set: The cardinality of an empty set is zero
  i.e. \(S = \emptyset\)
- Singleton (Unit) Set: Set that contains only one element
- Equal Set: Two sets that contains the same elements
- Equivalent Set: Two sets that have the same cardinality
- Overlapping Set: Two sets that have at least one element in common
- Disjoint Set: Two sets that do not have any elements in common

** Operations on Classical Sets
:PROPERTIES:
:CUSTOM_ID: operations-on-classical-sets
:END:
The following are some basic operations that can be done on sets
#1:

- Union: \(A = \{10,11,12,13\}\) and \(B = \{13,14,15\}\) then
  \(A \cup B = \{10,11,12,13,14,15\}\)

- Intersection:
  - [[https://www.tutorialspoint.com/fuzzy_logic/images/intersection_operation.jpg][Intersection]]
- Difference / Relative Complement: The set difference of sets \(A\) and
  \(B\) is the set of elements of which are only in \(A\) but not in
  \(B\).
  - [[https://www.tutorialspoint.com/fuzzy_logic/images/relative_complement_operation.jpg]]
- Complement of a set:
  - [[https://www.tutorialspoint.com/fuzzy_logic/images/complement_of_set.jpg]]
- Cartesian Product / Cross Product: The set of all possible ordered
  pairs.
  - Example: \(A=\{a,b\}\) and \(B=\{1,2\}\) then
    \(A\times B = \{(a,1),(a,2),(b,1),(b,2)\}\)

** Properties of Classical Sets
:PROPERTIES:
:CUSTOM_ID: properties-of-classical-sets
:END:
- Commutative Property
  - \(A \cup B = B \cup A\)
  - \(A \cap B = B \cap A\)
- Associate Property
  - \(A \cup (B \cup C) = (A \cup B) \cup C\)
  - \(A \cap (B \cap C) = (A \cap B) \cap C\)
- Distributive Property
  - \(A\cup \left ( B\cap C \right ) = \left ( A\cup B \right )\cap \left ( A\cup C \right )\)
  - \(A\cap \left ( B\cup C \right ) = \left ( A\cap B \right )\cup \left ( A\cap C \right )\)
- Idempotency Property
  - \(A \cup A = A\)
  - \(A \cap A = A\)
- Identity Property: For set \(A\) and universal set \(X\), this
  property statesA
  - \(A \cap X = A\)
  - \(A \cup X = X\)
- Transitive Property
  - If \(A \subseteq B \subseteq C\), then \(A \subseteq C\)
- Involution Property
  - \(\overline{{\overline{A``` = A\)
- De Morgan's Law
  - \(\overline{A\cap B} = \overline{A} \cup \overline{B}\)
  - \(\overline{A\cup B} = \overline{A} \cap \overline{B}\)
