Set Theory

A set is an unordered collection of different elements. For example

{{$ A = {1,2,3,4,5} }}$

is a set |#1.

Representation of Sets

Sets are typically represented in two ways

  • Roster or Tabular Form

{{$ A = {1,2,3,4} }}$

  • Set Builder Notation

{{$ A = {x:p(x)} }}$

where is some function that describes the set |#1.

Cardinality

Cardinality defines the number of elements in a set |#1.
