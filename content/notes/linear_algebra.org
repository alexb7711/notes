** Conjugate Matrix
:PROPERTIES:
:CUSTOM_ID: conjugate-matrix
:END:
The conjugate matrix is obtained by taking the /complex conjugate/ of
each element of the matrix. If we are given a complex variable
\(z = x + iy\), the complex conjugate of it is represented as

\[
\bar{z} = x - iy
\]

Therefore, the conjugate matrix does this for every element in the
matrix.
