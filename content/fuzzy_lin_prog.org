* State of the Art
:PROPERTIES:
:CUSTOM_ID: state-of-the-art
:END:
The main objective of fuzzy linear programming is to find the best solution possible with imprecise, vague, uncertain or
incomplete information. In literature, most of the fuzzy linear programming models deal with the fuzziness related to
the function(s)' coefficients, technological coefficients, and the right hand side values of the constraints.

Because the decision making process in crisp solutions expect exact conditions, presenting a reasonable range of values
for the decision variables can be comparitavely better than the currently available crisp solution. Fuzzy solutions may
serve the regions containing potential satisfactory solutions around the optimal solution. Therefore, even with fuzzy
linear programs, the final decisions can be the same as the crisp.

Fully fuzzy linear programs have been an attractive topic to reseachers for years. However, the literature on fully
fuzzy linear programming is not rich as there are few studies available on this topic.

** Preliminaries

#+caption: Definition 2.1
#+begin_quote
Let $X$ be a classical set of objects. Then, the set of ordered pairs $\tilde{A} = \{(x, \membership) : x \in X\}$, where
$\fuzzy_set : \rightarrow [0, 1]$, is called a fuzzy set in $X$. the evaulation function $\membership$ is called the membership
function.
#+end_quote

#+caption: Definition 2.2
#+begin_quote
Let $\tilde{A}$ be a fuzzy set in $X$ and $\lambda \in [0,1]$ be a real number. Then, a classical set $A^{\lambda} \{ x \in X :
\membership \ge \lambda \}$ is called an $\lambda$-cut or parametric form of $\tilde{A}$.
#+end_quote

#+caption: Definition 2.3
#+begin_quote
A fuzzy set $\tilde{A} = \{ (x, \membership) : x \in X \}$ is called a normalized set iff $\text{Supremum\{\membership\} = 1$
#+end_quote

#+caption: Definition 2.4
#+begin_quote
A fuzzy set $\tilde{A}$ is called a convex fuzzy set iff $\mu_{\tilde{A}} (\alpha x_1 + (1 - \alpha) x_2) \ge \text{min} \{\mu_{\tilde{A}}(x_1), \mu_{\tilde{A}}(x_2)\}, \forall x_1 , x_2 \in X, \alpha \in [0,1]$.
#+end_quote

#+caption: Definition 2.5
#+begin_quote
A convex normalized set $\tilde{A} = \{ (x, \membership) : x \in X \}$ is called a fuzzy number iff $\membership$ is piecewise continuous in $X$.
#+end_quote

#+caption: Definition 2.6
#+begin_quote
A fuzzy number $\tilde{A}$ is said to be a non-negative fuzzy number iff $\membership = 0, \forall x < 0$
#+end_quote

#+caption: Definition 2.7
#+begin_quote
A fuzzy number $\tilde{A}$ defined on the universal set of real numbers $\mathbb{R}$, denoted as $\tilde{A} = (a,b,c,d)$, is said to be a trapezoidal fuzzy number if its membership function, $\membership$, is given by

\begin{equation}
\membership =
\begin{cases}
\frac{(x-a)}{(b-a)} & a \le x < b \\
1                   & b \le x > c \\
\frac{(x-d)}{(c-d)} & c < x \le d  \\
0                   & \text{otherwise}
\end{cases}
\end{equation}
#+end_quote

#+caption: Definition 2.8
#+begin_quote
Let $\tilde{A} = (a,b,c,d)$ be a trapezoidal fuzzy number. Then, its $\lambda$-cut $A^{\lambda}$, is defined as follows: $A^{\lambda} = [a + (b-a)\lambda, d - (d-c)\lambda], 0 \le \lambda \le 1$
#+end_quote

#+caption: Definition 2.9
#+begin_quote
A trapezoidal fuzzy number is said to be a symmetric trapezoidal fuzzy number iff $b-a = d-c$, otherwise $\tilde{A}$ is said to be an asymmetric trapezoidal fuzzy number.

a symmetric trapezoidal can be denoted as $(b,c,\simga)$, where $[b,c]$ is the core and $\simga = b-a = d-c$ is the spead of the symmetric trapezoidal fuzzy number $\tilde{A}$.
#+end_quote

#+caption: Definition 2.10
#+begin_quote
A trapezoidal fuzzy number $\tilda{A} = (a,b,c,d)$ is said to be a non-negative trapezoidal fuzzy number iff $a \ge 0$ and is said to be non-positive trapezoidal fuzzy numbef iff $a \le 0$
#+end_quote

#+caption: Definition 2.11
#+begin_quote
A trapezoidal fuzzy number is said to be unrestricted trapezoidal fuzzy number iff $a$ is a real number
#+end_quote

#+caption: Definition 2.11
#+begin_quote
Two trapezoidal fuzzy numbers $\tilde{A}_1$ and $\tilde{A}_2$ are said to be equal iff $a_1 = a_2$, $b_1 = b_2$, $c_1 = c_2$, and $d_1 = d_2$.
#+end_quote

* Misc Papers
Fuzzy linear programming introduces the idea of uncertainty into the model's parameters. This allows for adaptability of
plans or "fuzziness" to the solution of the linear program. By doing so, this allows for a more robust model that is
guaranteed to result in an efficient solution #1 #3 (TODO: fix ref). Fuzzy LP is based off [[file:fuzzy_theory.org][fuzzy theory]] which is the
basis that allows for the uncertainty.

When a Linear Program's (LP) parameters and decision variables are to be modeled as fuzzy, then the LP is said to be a
Fully Fuzzy Lineal Programming Problem (FFLP) #1 In other cases when only part of the LP is fuzzy, it is known as a
Fuzzy Linear Program (FLP). There are many methods to solve FLP's and FLLP's. Most of them convert the FFLP into a LP
#3.

In #3, Zimmerman describes 3 approaches to "single out" one specific solution from the set of efficient solutions which
qualifies as "optimal"

- The utility approach
- Goal Programming
- Interactive Approaches

The first two assumes that the decision maker can specify their "preference-function" with respect to the combination of
the individual objective functions in advance as either a sum of weights (utilities) or distance functions (distance from
ideal solution). The third uses only local information in order to arrive at an acceptable compromise solution. We new
introduce the fuzzy approach.

** Fuzzy Objective
:PROPERTIES:
:CUSTOM_ID: fuzzy-objective
:END:
The fuzzy objective function is characterized by its [[file:fuzzy_theory.org][membership function]], same with its constraints. The "decision" in a
fuzzy environment, much like its crips brother, can be viewed as the intersection of fuzzy constraints and fuzzy
objective functions. This is implying that the relationship between constraints and the objective functions is fully
symmetric (there is no longer a difference between the former and latter) #3.

** Fuzzy Constraints
:PROPERTIES:
:CUSTOM_ID: fuzzy-constraints
:END:
As stated previously, fuzzy constraints tolerate violations in the accomplishment of the constraints. The constraint can
be represented by

#+begin_latex latex
\begin{equation}
a_i x \lesssim b_i
\end{equation}
#+end_latex

Where \(a\) and \(b\) modeled using a [[file:fuzzy_theory.org][membership function ]] as described in #2 and #3.

#+begin_latex latex
\begin{equation}
\mu_i (x) =
\begin{cases}
    1          & \text{if } a_i x \leq b                        \\
    f_i(a_i x) & \text{if } b_i \leq a_i x \leq b_i + \bar{b}_i \\
    0          & \text{if } a_i x \geq b_i + \bar{b}_i          \\
\end{cases}
\end{equation}
#+end_latex

A fuzzy solution is found from the solution of the [[file:parametric_lin_prog.org][parametric linear #2.

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    \text{max } xc    \\
    Ax \leq g(\alpha) \\
    x \geq 0          \\
    \alpha \in (0,1]
\end{array}
\end{equation}
#+end_latex

where \(g(\alpha)\) is a column vector defined by the inverse functions of the \(f_i\).

Zimmerman proposes that given a LP:

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    cx \lesssim z_0 \\
    Ax \lesssim b   \\
    x \gtrsim 0
\end{array}
\end{equation}
#+end_latex

with a linear [[file:fuzzy_theory.org][membership function]], the solution can be of the form #3.

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    f(x) = 1 - \frac{(Bx)_i - b_i'}{\bar{b}_i} \\
    \lambda \leq \frac{b_i'}{\bar{b}_i} - (\frac{B}{\bar{b}} x)_i
\end{array}
\end{equation}
#+end_latex

where \(f(\cdot)\) defines the linear membership function and \(\lambda\) is the developed linear constraint. \(B\) is the
constraint matrix from \(Ax \leq b\) augmented with the objective equations. Similarly \(b'\) is \(b\) augmented with upper
bounds for the objective equations, and \(\bar{b}\) are subjectively chosen constraints off addmissible violations #3.

** Fuzzy Coefficients
:PROPERTIES:
:CUSTOM_ID: fuzzy-coefficients
:END:
If the decision maker does not know exactly the values of the coefficients taking part in the problem, fuzzy numbers can
be used to represent this uncertainty #2.

#+begin_latex latex
\begin{equation}
\sum_{j=1}^n \tilde{a}_{ij} x_j \leq \tilde{b}_i
\end{equation}
#+end_latex

Given a similar model to solve as before the model for fuzzy coefficients can be defined as

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    \text{max } cx                                                  \\
    [(1-1/2h)(a_i + \bar{a}_i) + 1/2h(a_i - \underline{a}_i)]x \leq \\
    (1-1/2h)(b_i + \bar{b}_i) + 1/2h(b_i - \underline{b}_i),        \\
    [1/2h(a_i + \bar{a}_i) + (1-1/2h)(a_i - \underline{a}_i)]x \leq \\
    1/2h(b_i + \bar{b}_i)+(1-1/2h)(b_i - \underline{b}_i),          \\
    x \geq 0
\end{array}
\end{equation}
#+end_latex

where \(h \in (0,1]\) is a level or degree of optimism being specified by the decision maker a priori. \((a + \bar{a})\)
is the upper bound and \((a - \underline{a})\) is the lower bound. Furthermore it is assumed that the upper bound of
\(a\) is greater than \(b\) and similarly lower bound of \(a\) is greater than \(b\).

* Formulating FFLP
:PROPERTIES:
:CUSTOM_ID: formulating-fflp
:END:
This section contains a list of different ways of formulating LP's.

** A General FLP Model
:PROPERTIES:
:CUSTOM_ID: a-general-flp-model
:END:
Consider

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    \text{max } cx \\
    Ax \lesssim b  \\
    x \geq 0
\end{array}
\end{equation}
#+end_latex

where \(a\) is a matrix of fuzzy numbers and \(b\) is a column vector of
fuzzy numbers. We can then create an ordering via
[[file:fuzzy_theory.org][ranking functions]] and state the "fuzziness"
with the following constraint:

#+begin_latex latex
\begin{equation}
Ax < b + (1-\alpha)
\end{equation}
#+end_latex

Where if \(\alpha=1\) then the constraint is completely verified and if
\(\alpha = 0\) then the variable is fully "fuzzy".

*** Nasseri's Method
:PROPERTIES:
:CUSTOM_ID: nasseris-method
:END:
Given an FFLP:

#+begin_latex latex
\begin{equation}
    max\; \sum_{j=1}^n \tilde{c}_j \tilde{x}_j \\

subject to
    \sum_{j=1}^n \tilde{a}_{ij} \tilde{x}_j \leq \tilde{b}_i\; \forall i = 1 - m \\
\end{equation}
#+end_latex

where \(\tilde{c_j}\), \(\tilde{a_{ij}}\), \(\tilde{b_j}\) are parameters
and \(\tilde{x_j}\) are non-negative fuzzy numbers #1. Because these are fuzzy numbers, we can
represent them as such:

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    \tilde{c}_j    = (c1_j, c2_j, c3_j)          \\
    \tilde{a}_{ij} = (a1_{ij}, a2_{ij}, a3_{ij}) \\
    \tilde{b}_j    = (b1_j, b2_j, b3_j)          \\
    \tilde{x}_j    = (x1_j, x2_j, x3_j)          \\
\end{array}
\end{equation}
#+end_latex

Nasseri then proposes that the FFLP can be rewritten as a LP in the
following manner:

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    max\; R\Big(\sum_{j=1}^n (c1_j, c2_j, c3_j)(x1_j, x2_j, x3_j)\Big) \\
    \text{Subject to}\\
    \sum_{j=1}^n a1_{ij}x1_j \leq b1_j \\
    \sum_{j=2}^n a2_{ij}x2_j \leq b2_j \\
    \sum_{j=3}^n a3_{ij}x3_j \leq b3_j \\
    x2_j - x1_j \geq 0 \\
    x3_j - x2_j \geq 0 \\
\end{array}
\end{equation}
#+end_latex

*** Zimmermans's Methods
:PROPERTIES:
:CUSTOM_ID: zimmermanss-methods
:END:
Zimmerman proposes two methods in his work:

- FLP using the 'min' operator
- FLP using the dot product operator

**** min operator
:PROPERTIES:
:CUSTOM_ID: min-operator
:END:
Given a FLP with objective function(s) \(c^T x\) that is subject to
constraints \(Ax \leq b\). A membership function can be written such
that

#+begin_latex latex
\begin{equation}
\begin{cases}
    0                        & \text{for } (A'x)_i \leq b'_i            \\
    \frac{(A'x)_i b'_i}{d_i} & \text{for b_i < (A'x)_i \leq b'_i + d_i} \\
    1                        & \text{for } (A'x)_i > b'_i + d_i
\end{cases}
\end{equation}
#+end_latex

Where this draws a line from \(z(-b'_i) = 0\) to \(z(d_i) = 1\).
Constraints can then be made using the form

#+begin_latex latex
\begin{equation}
\lambda = \frac{b'_i}{d_i} - (\frac{A}{d_i}x)_i
\end{equation}
#+end_latex

which is appended to the set of constraints.

**** product operator
:PROPERTIES:
:CUSTOM_ID: product-operator
:END:
This method uses the same membership function

#+begin_latex latex
\begin{equation}
\begin{cases}
    0                        & \text{for } (A'x)_i \leq b'_i            \\
    \frac{(A'x)_i b'_i}{d_i} & \text{for b_i < (A'x)_i \leq b'_i + d_i} \\
    1                        & \text{for } (A'x)_i > b'_i + d_i
\end{cases}
\end{equation}
#+end_latex

Rather than append to the constraints and maximize or minimize
\(\lambda\), the objective function is updated with

#+begin_latex latex
\begin{equation}
\text{max } \prod_{i = 1}^K = \frac{(c^i)^T x - p_i}{d_i}
\end{equation}
#+end_latex

where \(c^i\) is the vector to be multiplied with \(x\). The constraints
are left the same.

** Zimmerman's Product Example
:PROPERTIES:
:CUSTOM_ID: zimmermans-product-example
:END:
A company manufactures two products, 1 and 2 on given capacities.

- Product 1:
  - profit of 2$
  - import cost of of 1$
- Product 2:
  - profit of 1$
  - profit of 2$ (exported)

As such, there are two goals

1) Profit maximization
2) Maximum improvement of the balance of trade (maximum difference of
   exports minus imports)

As an example, the membership function for \(z_1\) is shown in the
figure below

[[file:img/z1_membership_func.png][Membership function]]

and the problem is written as

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    \text{max } Z(x) =
    \begin{bmatrix}
        -1 & 2 \\
        2  & 1
    \end{bmatrix}
    \begin{bmatrix}
        x1 \\
        x2
    \end{bmatrix} \\
    -x_1 + 3x_2 \leq 21 \\
    x_1 + 3x_2 \leq 27  \\
    4x_1 + 3x_2 \leq 45 \\
    3x_1 + x_2 \leq 30  \\
    x1,x2 \geq 0
\end{array}
\end{equation}
#+end_latex

Using the method proposed by Zimmerman (as described in
Fuzzy Constraints, one yields the following formulation

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    \text{max } \lambda \\
    \lambda \leq -0.05582x_1 + 0.117x_2 + 0.1764 \\
    \lambda \leq 0.1429x_1 + 0.0714x_2 - 0.5     \\
    21 \geq -x1 + 3x_2                           \\
    27 \geq x1 + 3x_2                            \\
    45 \geq 5x_1 + 3x_2                          \\
    30 \geq 3x_1 + x_2                           \\
    x \geq 0
\end{array}
\end{equation}
#+end_latex

Where the maximum "overval satisfaction" of \(\lambda = 0.74\) is
achived with \(x = [5.03, 7.32]^T\).

** Fuzzy Berth Allocation Problem (BAP) Example
:PROPERTIES:
:CUSTOM_ID: fuzzy-berth-allocation-problem-bap-example
:END:
In the [[file:berth_allocation_problem.org][BAP]], arrivals of each
vessel are highly uncertain, meaning vessels can arrive earlier or later
than they were scheduled to arrive. Having to frequently review the
berthing plan is not desirable from a planning of resources point of
view. A FFLP for the BAP problem to minimize the total time can be
expressed as

#+begin_latex latex
\begin{equation}
min\; \sum_{i\in V} (\tilde{m}_i - \tilde{a}_i)
\end{equation}
#+end_latex

Such that

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    \tilde{m}_i \geq \tilde{a}_i                               \\
    p_i + l_i \leq L                                           \\
    p_i + l_i \leq p_j + M(1-z_{ij}^x)                         \\
    \tilde{m}_i + \tilde{h}_i \leq H                           \\
    \tilde{m}_i + \tilde{h}_i \leq \tilde{m}_j + M(1-z_{ij}^x) \\
    z_{ij}^x + z_{ji}^x + z_{ij}^y + z_{ij}^y + \geq 1         \\
    z_{ij}^x, z_{ij}^y \in \{0,1\}
\end{array}
\end{equation}
#+end_latex

where the parameters are

- \(L\) : Total length of the quay
- \(H\) : Planning horizon
- \(V\) : Set of incoming vessels
- \(a_i\) : Arrival time at port
- \(l_i\) : Vessel length
- \(h_i\) Handling time of the vessel (\(\tilde{h} = (h, h, h)\)
  - This value is considered a /singleton/?

and the decision variables are

- \(m_i\) : Berthing time of the vessel
- \(p_i\) : Berthing position of the vessel
- \(z_{ij}^x\) : Indicates if vessel \(i\) is to the left of vessel
  \(j\) at berth
- \(z_{ij}^y\) : Indicates that the berthing time of vessel \(i\) is
  before that of \(j\)

Using Nasseri's Method in #1 the BAP can be
described as

#+begin_latex latex
\begin{equation}
min\; \sum_{i\in V} (m1_i - a3_i, m2_i - a2_i, m3_i - a1_i)
\end{equation}
#+end_latex

Subject to

#+begin_latex latex
\begin{equation}
\begin{array}{l}
    {m}1_i \geq {a}1_i                                 \\
    {m}2_i \geq {a}2_i                                 \\
    {m}3_i \geq {a}3_i                                 \\
    p_i + l_i \leq L                                   \\
    p_i + l_i \leq p_j + M(1-z_{ij}^x)                 \\
    {m}1_i + {h}1_i \leq H                             \\
    {m}2_i + {h}2_i \leq H                             \\
    {m}3_i + {h}3_i \leq H                             \\
    {m}1_i + {h}_i \leq {m}1_j + M(1-z_{ij}^x)         \\
    {m}2_i + {h}_i \leq {m}2_j + M(1-z_{ij}^x)         \\
    {m}3_i + {h}_i \leq {m}3_j + M(1-z_{ij}^x)         \\
    z_{ij}^x + z_{ji}^x + z_{ij}^y + z_{ij}^y + \geq 1 \\
    z_{ij}^x, z_{ij}^y \in \{0,1\}
\end{array}
\end{equation}
#+end_latex
