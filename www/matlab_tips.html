Pivoting Function

The original code can be found here |#1:

function R = pivot(M, r, c)
    % Get matrix dimensions
    [d, w]  = size(M);

    % Initialize to appropriate size
    R       = zeros(d, w);

    % Copy row r, normalizing M(r,c) to 1
    R(r, :) = M(r, :) / M(r, c);

    % For all matrix rows
    for k = 1:d
        %content%lt;!-- Other than the row of interest r --%content%gt;
        if (k ~=r)
            %%content%lt;!-- Set it equal to the original matrix --%content%gt;
            %%content%lt;!-- minux a multiple of normalized row r, making --%content%gt;
            %%content%lt;!-- R(k,c) = 0 --%content%gt;
            R(k,:) = M(k,:) - M(k,c) * R(r,:)
        end % if
    end         % for
end                 % function
